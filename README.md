# Simple xdebug / mac / PhpStorm use case

## Build 

`docker-compose build`

## Start 

`docker-compose up -d`
`composer install`

## Run

Go to http://localhost:6700

## Configure PHPStorm

Follow instructions given by the JetBrains team https://www.jetbrains.com/help/phpstorm/configuring-xdebug.html

Use port 9999
